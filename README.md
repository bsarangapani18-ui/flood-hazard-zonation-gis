🌊 Flood Hazard Zonation (GIS Project)
📌 Overview

This project analyzes flood vulnerability using GIS techniques to support regional planning and risk management.

⚙️ Key Work
Prepared thematic layers (rainfall, slope, land use, drainage, soil, elevation)
Applied AHP-MCDA method for weighted analysis
Generated district-level flood hazard zonation map
Achieved Consistency Ratio = 0.04
📊 Output
Final flood hazard map showing low, medium, and high-risk zones
Spatial dataset suitable for planning and decision-making
🎯 Impact

Supports disaster management and land-use planning by identifying high-risk flood zones.

🛠️ Tools Used
ArcGIS 10.7
GIS Analysis Techniques
Remote Sensing Data
